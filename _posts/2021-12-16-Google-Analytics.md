## google analytics 추가법

1. google analytics에 가입한다.

2. 속성 추가를 눌러 github 사이트를 등록한다.

3. 생성된 ID를 복사하여 `_config.yml`에 적용한다.

```
google_analytics : ID
```

해당 코드를 추가한다.

4. google analytics에 접속하여 실시간으로 방문자가 변화하는 것을 관찰한다.
